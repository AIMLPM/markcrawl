# Security Policy

## Reporting a vulnerability

Please do not open a public issue for security-sensitive reports.

Instead, contact the maintainer privately with:

- a description of the issue
- steps to reproduce
- affected files or behavior
- any suggested mitigation

## Scope

This project is a crawler for public website content. Security-sensitive areas may include:

- unsafe URL handling
- file path handling
- unexpected redirects
- excessive resource use
- dependency vulnerabilities

## Supported versions

This repository is currently early-stage. Security fixes will generally be applied to the latest version on the default branch.
