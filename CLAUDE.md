# CLAUDE.md

Project-level instructions for AI assistants working on MarkCrawl.

## Project overview

MarkCrawl is a web crawler that produces clean Markdown for LLM/RAG pipelines.
It handles JavaScript-rendered pages, deduplicates content, and outputs
structured Markdown ready for chunking and embedding.
